media query는 우리가 조건을 추가할 수 있는 방법
코드의 조건을 적는 것
코드의 조건이 참이라면 해당 CSS를 실행하라는 것
emdia query 안에 원하는 CSS를 넣을 수 있다
element에 CSS를 적용시켜야만 사용 가능하다.
media query는 and를 넣어서 사용이 가능하다.
min-width and max-width를 넣어서 사용 가능, orientation도 사용 가능하다
다른 조건도 사용 가능한데, device-width로 사용 가능하다.
다른 media type도 있지만 자주 사용되지 않는다.
ex) print : 내가 스크린을 인쇄하고 싶을 때 사용한다.
