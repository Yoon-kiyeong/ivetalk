- index.html: 대부분의 웹 서버는 default로 해당 파일부터 찾도록 설정 됨
- 전체적인 틀을 마련해주는 VS: ! + Enter
- 주석 <!-- -->: Ctrl + /
- 중복 가능성을 피하는 클래스명 찾기: 부모__자식

- 코드 간략작성법
- 태그#id명: (tag id ="id명")(/tag)
- 태그class명: (tag class ="클래스명")(/tag)
- 태그 > 자식태그 -> (tag)(자식tag)(/자식tag)(/tag)
- 태그 * 숫자n -> (tag)(/tag)를 n번 작성
- 합쳐서 사용 가능
