web font
- link보다 import를 추천한다.
- import의 위치는 최상단
- 모든 폰트를 추가하면 웹사이트가 무거워진다.

css hack(justify-content 대신 사용가능)
- 레시피 같이 어디든 쓸 수 있다. 이상하지만 아무튼 작동
- 1. 상위 박스 : jsutify-content:center; -> 중앙으로 몰림
- 2. 내부 박스 범위 : width: 33% -> 왼쪽으로 몰려서 범위 벌어짐, 왼쪽 위치할 박스는 왼쪽에 붙어서 정렬됨
- 3. 중앙에 위치할 박스 : dispaly: flex; justify-content: center; -> 중앙에 위치할 박스만 중앙에 위치함
- 4. 오른쪽에 정렬할 박스 : *display*: flex, *justify-content*: flex-end; *align-items*: center; -> 오른쪽에 붙어서 정렬됨
